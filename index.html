OTP 인증

웹에서 OTP를 생성한 뒤, 금고 키패드에 고정 비밀번호 + OTP + #을 입력합니다.

------

남은 시간: 0초

제어

인증 안내

입력 예시: 1234 + OTP + #

OTP 유효시간은 3분이며, 사용 후 즉시 폐기됩니다.

OTP 없이 입력하면 위험도가 증가하고 LCD에 OTP 발급 안내가 표시됩니다.
